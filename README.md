# Maginus Lunch
The Maginus Lunch Ordering App.

A little side project for me exploring .Net Core 3.1, VSCode and Task-based APIs.

See [Maginus Lunch](https://martinpickering.com/posts/maginus-lunch/introduction) on [my personal site](https://martinpickering.com) for more information about this Project.

NOTE: I did start out in .Net5 Preview, but I found myself missing too many third party features to feel 
comfortable with that preview for this work as yet. I was particularly missing the MOQ Mocking Framework.
So, I reverted to .Net Core V3.1. Still worthwhile for me as until now I was still doing .Net Core V2.1.

I also introduced using Visual Studio 2019 rather than just VSCode. I'll still continue to use VSCode, its just
that sometimes there is only so much one can learn all at once yet the pressure to get stuff done still exists.
